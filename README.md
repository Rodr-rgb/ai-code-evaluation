# AI Code Evaluation Dataset

## Overview
This project evaluates AI-generated code for correctness, efficiency, and robustness.

## Focus Areas
- Bug detection
- Logical correctness
- Edge case handling
- Code quality

## Methodology
Each entry includes:
- Prompt
- Generated code
- Identified issues
- Score
- Improved version

## Use Case
Useful for training AI systems to generate high-quality code.
